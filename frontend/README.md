# Vesto Project – Plataforma de Controle Financeiro e Investimentos

## Sobre o Projeto
O **Vesto Project** é uma plataforma web em fase de desenvolvimento utilizando **Django REST Framework (backend)** e **React.js (frontend)** para gerenciamento de **finanças pessoais e investimentos**.  
O objetivo é oferecer uma aplicação simples e intuitiva que permita registrar **despesas, ganhos e investimentos**, organizando as movimentações financeiras em um só lugar.
A ideia do desenvolvimento é começar com funcionalidades simples e, ao longo do tempo, aprimorar a plataforma. O objetivo principal é reunir em um único lugar o gerenciamento das minhas finanças pessoais, ao mesmo tempo em que aprofundo meus conhecimentos nas tecnologias utilizadas.

---

## Funcionalidades

### Despesas e Ganhos
- Registrar **despesas** e **receitas** com:
  - Valor
  - Data
  - Descrição
  - Categoria
  - Método de pagamento
- Visualizar **histórico de movimentações** em tabela responsiva.
- Categorização dinâmica (categorias de **despesa** e **ganho**).
- Excluir transações.

### Investimentos
- Registrar investimentos do tipo:
  - **Ação**
  - **Fundo Imobiliário**
  - **Renda Fixa**
  - **EUA (internacional)**
- Campos disponíveis:
  - Tipo
  - **Ativo (sigla)**
  - Valor
  - Cotas
  - Data
- Histórico de investimentos com listagem.
